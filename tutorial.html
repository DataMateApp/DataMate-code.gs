Tutorial: How to Use the Enhanced Form Builder App

This app creates a customizable form with advanced fields. Use the sidebar ("Form Builder > Open Form Builder") or edit the A9:E range directly in the FormSetup sheet.

Step 1: Configure Settings

Set the following in the FormSetup sheet:

  • B3: Form Name (e.g., "Customer Survey") - Defines the title of your form.
  • B4: Target Sheet (e.g., "Responses") - The sheet where form data will be stored.
  • B5: Submission Type ("SingleCell" or "TableRow") - SingleCell writes data to specific cells; TableRow adds each submission as a new row.

Step 2: Add Fields

Add fields using the sidebar or edit A9:E directly:

  • A: Field Name (e.g., "Name") - The label for the field.
  • B: Target Sheet (e.g., "Responses") - Must match B4 or specify another sheet.
  • C: Target Cell/Column (e.g., "A1" or "A") - Where data is stored (A1 for SingleCell, column letter like A for TableRow).
  • D: Field Type (e.g., "Text") - Defines the input type (see below).
  • E: Options (e.g., "Option1,Option2" or "=Sheet1!A:A") - Additional settings like dropdown options or formulas.

Step 3: Field Types

Below are all supported field types with instructions:

Text

A single-line text input.

  • E: Leave blank.
  • Example: ["Name", "Responses", "A", "Text", ""]

Dropdown

A dropdown menu with selectable options.

  • E: Comma-separated options (e.g., "Yes,No") or a sheet range (e.g., "=Sheet1!A:A").
  • Example: ["Choice", "Responses", "B", "Dropdown", "Red,Blue,Green"]

MultiSelect

A dropdown allowing multiple selections.

  • E: Comma-separated options or a sheet range.
  • Example: ["Interests", "Responses", "C", "MultiSelect", "Tech,Art,Sports"]

Date

A date picker input.

  • E: Leave blank.
  • Example: ["Event Date", "Responses", "D", "Date", ""]

Time

A time picker input.

  • E: Leave blank.
  • Example: ["Event Time", "Responses", "E", "Time", ""]

Number

A numeric input field.

  • E: Leave blank.
  • Example: ["Quantity", "Responses", "F", "Number", ""]

Checkbox

A single checkbox (true/false).

  • E: Leave blank.
  • Example: ["Urgent", "Responses", "G", "Checkbox", ""]

Radio

Radio buttons for single-choice selection.

  • E: Comma-separated options.
  • Example: ["Priority", "Responses", "H", "Radio", "Low,Medium,High"]

Textarea

A multi-line text input.

  • E: Leave blank.
  • Example: ["Comments", "Responses", "I", "Textarea", ""]

Email

An email input with validation.

  • E: Leave blank.
  • Example: ["Email", "Responses", "J", "Email", ""]

StarRating

A 5-star rating input.

  • E: Leave blank (fixed 1-5 scale).
  • Example: ["Rating", "Responses", "K", "StarRating", ""]

RangeSlider

A slider for selecting a value within a range.

  • E: "min,max,step" (e.g., "0,100,5").
  • Example: ["Satisfaction", "Responses", "L", "RangeSlider", "0,100,5"]

FileUpload

Allows users to upload files (images or PDFs, max 6MB).

  • E: Leave blank.
  • Behavior: Files are uploaded to Google Drive, and the form stores a link.
  • Example: ["Attachment", "Responses", "M", "FileUpload", ""]

Conditional

A field that appears based on another field's value.

  • E: "FieldName=Value" (e.g., "Urgent=Yes").
  • Example: ["Reason", "Responses", "N", "Conditional", "Urgent=Yes"]

Calculated

A read-only field that calculates a value.

  • E: "FieldName=Formula" (e.g., "Total=Quantity*10").
  • Example: ["Total", "Responses", "O", "Calculated", "Total=Quantity*10"]

Signature

A canvas for drawing a signature.

  • E: Leave blank.
  • Behavior: Saves as a PNG link in Google Drive.
  • Example: ["Signature", "Responses", "P", "Signature", ""]

Geolocation

Captures latitude and longitude coordinates.

  • E: Leave blank.
  • Behavior: Users click a button to share their location.
  • Example: ["Location", "Responses", "Q", "Geolocation", ""]

ProgressBar

Displays a static progress bar.

  • E: A number (0-100) for the progress value.
  • Example: ["Progress", "Responses", "R", "ProgressBar", "75"]

Captcha

A simple verification field (fixed to "3 + 5").

  • E: Leave blank.
  • Behavior: Users must enter "8" to pass.
  • Example: ["Verification", "Responses", "S", "Captcha", ""]

Image

Displays a static image.

  • E: A Google Drive link (e.g., "https://drive.google.com/uc?id=FILE_ID").
  • How to Add:
    1. Upload an image to Google Drive.
    2. Share it as "Anyone with the link."
    3. Use format: https://drive.google.com/uc?id=FILE_ID.
  • Example: ["Product Image", "Responses", "T", "Image", "https://drive.google.com/uc?id=1a2b3c4d5e6f7g8h9i0j"]

Video

Displays a playable video.

  • E: A direct video URL (e.g., MP4 link).
  • Example: ["Product Video", "Responses", "U", "Video", "https://www.w3schools.com/html/mov_bbb.mp4"]

ImageLink

Allows users to input an image URL.

  • E: Leave blank.
  • Example: ["Upload Image URL", "Responses", "V", "ImageLink", ""]

VideoLink

Allows users to input a video URL.

  • E: Leave blank.
  • Example: ["Upload Video URL", "Responses", "W", "VideoLink", ""]

StaticText

Displays static text.

  • E: The text to display.
  • Example: ["Welcome", "Responses", "X", "StaticText", "Please fill out this form."]

Table

Displays a table from a sheet range.

  • E: A range (e.g., "Sheet2!A1:C5").
  • Example: ["Sales Data", "Responses", "Y", "Table", "Sheet2!A1:C5"]

Container

A section to group fields with optional custom styling.

  • E: Inline CSS (e.g., "background: #f0f0f0; padding: 10px;") or leave blank for default styling.
  • Behavior: Groups all following fields until another Container or Footer.
  • Example: ["Section 1", "Responses", "Z", "Container", "background: #f0f0f0; border: 2px solid #ccc;"]
  • Tip: Place before the fields you want to group.

Header

A styled header section at the top or within the form.

  • E: Text to display in the header.
  • Example: ["Form Header", "Responses", "AA", "Header", "Welcome to Our Survey"]
  • Tip: Use at the start of A9:E for a top header.

Footer

A styled footer section at the bottom or within the form.

  • E: Text to display in the footer.
  • Example: ["Form Footer", "Responses", "AB", "Footer", "© 2025 Your Company"]
  • Tip: Place at the end of A9:E for a bottom footer.

Step 4: Preview & Deploy

Preview via "Form Builder > Preview Form". Deploy via Apps Script:

  • Go to the Apps Script editor.
  • Click "Deploy > New Deployment > Web App."
  • Set "Who has access" to "Anyone" or as needed, then deploy.
  • Copy the generated URL.

Step 5: Collect Data

Share the web app URL with respondents. Data is stored in the target sheet:

  • SingleCell: Overwrites specified cells.
  • TableRow: Adds new rows (creates new sheets if over 40,000 rows).
  • FileUpload/Signature: Files save to Google Drive with links in the sheet.

Instructions

1. Open your spreadsheet and locate or create the FormSetup sheet.
2. Paste the template above starting at A9.
3. Adjust column E (Dropdown Options) as needed (e.g., customize dropdowns, add more history/reference sections).
4. Use the "Preview Form" menu option to test, or deploy as a web app via the Apps Script editor.

Limitations

- Repeatable Sections: This template includes one instance of Employment History and References. To add more, duplicate the relevant rows in FormSetup with unique field names (e.g., "Employer Name 2").
- Formatting: Use the Container field type to group sections visually, and adjust styles in column E.
